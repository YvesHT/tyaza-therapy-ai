# TYAZA - Therapeutic AI Companion

A compassionate AI therapy assistant with voice capabilities.

## ? Features

- ?? **Voice Input** - Speak naturally, AI listens
- ?? **Natural TTS** - Warm, therapeutic voices
- ?? **Empathetic Responses** - Concise by default, deeper when needed
- ?? **PWA Ready** - Install as app on phone/desktop
- ?? **Privacy First** - Your API key saved locally in browser
- ?? **Session History** - Past conversations saved

## ?? Live Demo

[https://tyaza-therapy.netlify.app](https://tyaza-therapy.netlify.app)

## ?? Setup

1. Get free API key from [OpenRouter.ai](https://openrouter.ai)
2. Enter key once - saved forever in your browser
3. Click microphone to start speaking
4. Type or speak - TYAZA responds with empathy

## ??? Tech Stack

- HTML5 / TailwindCSS
- Web Speech API (STT/TTS)
- OpenRouter AI (Free models)
- LocalStorage for persistence
